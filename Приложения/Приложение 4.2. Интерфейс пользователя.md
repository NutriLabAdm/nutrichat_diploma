# Приложение 4.2. Интерфейс пользователя

## Введение

Приложение содержит описание и скриншоты основных экранов пользовательского интерфейса платформы Нутричат.

## Основные экраны

### 4.2.1. Главный дашборд

Основной экран после входа в систему:
- Краткая статистика дня (калории, БЖУ)
- Быстрые действия (добавить еду, открыть чат)
- Прогресс по целям
- Рекомендации AI

### 4.2.2. Дневник питания

Экран ведения дневника питания:
- Список приёмов пищи (завтрак, обед, ужин, перекусы)
- Добавление продуктов (ручной ввод, сканирование)
- Итого за день с графиком

### 4.2.3. AI-консультант (NutriChat)

Диалоговый интерфейс с AI-консультантом:
- Чат с персонализированными рекомендациями
- Контекст из Digital Twin пользователя
- История диалога

### 4.2.4. Harvard Plate Assessment

Инструмент оценки соответствия питания рекомендациям Harvard:
- Визуальная оценка тарелки
- Баланс групп продуктов
- Рекомендации по улучшению

### 4.2.5. NDI (Nutrient Density Index)

Индекс плотности нутриентов:
- Оценка качества рациона
- Дефициты микронутриентов
- Рекомендации по обогащению

### 4.2.6. Red Flags (Красные флаги)

Система выявления проблемных паттернов:
- Признаки расстройств пищевого поведения
- Нутритивные дефициты
- Поведенческие триггеры

### 4.2.7. Перерасчёт (Recalc)

Функция перерасчёта рекомендаций:
- Изменение целей
- Корректировка макросов
- Адаптация к новым условиям
