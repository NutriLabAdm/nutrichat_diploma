## Дизайн исследования (дополнение)

### Генерация аватаров из интервью

**Процесс:**

```
1. Интервью с пользователем
       ↓
2. Транскрибирование (STT)
       ↓
3. Выявление ключевых факторов:
   - Готовность платить (фразы: "ссылка на оплату", "перезвоните")
   - Барьеры (время, деньги, здоровье)
   - Мотивация (высокая/средняя/низкая)
   - Цели (похудеть, ЗОЖ, набор массы)
       ↓
4. Создание аватара с атрибутами
```

**Ключевые факторы из интервью:**

| Фактор | Индикаторы из текста |
|--------|---------------------|
| Готовность платить | "ссылка на оплату", "сколько стоит", "abonement" |
| Нет готовности | "перезвоните", "дорого", "подумаю" |
| Барьеры | "нет времени", "не могу", "проблемы" |
| Мотивация | "хочу", "надо", "нужно" |

### Структура эксперимента

**Распределение аватаров:**

- Всего: 40 аватаров
- Группа A (стандарт): 6 интервенций
- Группа B (AI): 14 интервенций

**Интервенции:**

| Группа A | Группа B |
|----------|----------|
| weight_tracking | + goal_setting |
| meal_logging | + progress_cheering |
| calorie_counter | + barrier_analysis |
| water_intake | + ai_recommendations |
| exercise_tracking | + персонализация |
| sleep_analysis | + мотивационные |

### Статистическая модель влияния

Каждая интервенция влияет на LTV через матрицу:

```
LTV = Base × Σ(Интервенция × Модулятор × Шум)
```

**Модуляторы:**
- Возраст: <40 (+10%), >60 (-20%)
- Мотивация: HIGH (+20%), LOW (-40%)
- Барьеры: time (-15%), finances (-25%), health (-20%)

### Запуск экспериментов

1. **Эксперимент 1:** Базовое сравнение A vs B
2. **Эксперимент 2:** Влияние мотивации
3. **Эксперимент 3:** Значимость каждой интервенции
4. **Эксперимент 4:** Uplift modeling валидация

**Цель:** Выявить статистически значимые интервенции для фокусировки ресурсов.
