## 5.1.1. Формализованная модель

**Базовая формула отклика:**

```
R(i, t) = f(A(t), I(i), C(t-1))

где:
- R(i, t) — реакция на интервенцию i в момент t
- A(t) — атрибуты Twin в момент t
- I(i) — параметры интервенции i
- C(t-1) — история контекста
- f — функция отклика модели
```

**Вероятностная модель отклика:**

```python
class TwinResponseModel:
    """
    Модель предсказания реакции Digital Twin на интервенцию
    """
    
    def __init__(self, twin_attributes: dict):
        self.motivation = twin_attributes['motivation']  # 0-1
        self.barriers = twin_attributes['barriers']       # список барьеров
        self.engagement_history = twin_attributes['history']  # история
        self.segments = twin_attributes['segments']       # сегменты
        
    def predict_response_probability(self, intervention: dict) -> float:
        """
        Предсказание вероятности положительного отклика
        """
        # Базовое влияние мотивации
        base_prob = self.motivation
        
        # Модификатор барьеров (каждый барьер снижает вероятность)
        barrier_modifier = 1.0
        for barrier in self.barriers:
            if intervention.get('addresses_barrier') == barrier:
                barrier_modifier *= 1.3  # Интервенция снимает барьер
            else:
                barrier_modifier *= 0.9  # Барьер остаётся
        
        # Модификатор истории вовлечённости
        recent_engagement = np.mean(self.engagement_history[-7:])
        engagement_modifier = 0.5 + recent_engagement * 0.5
        
        # Модификатор соответствия сегмента
        segment_modifier = 1.0
        if intervention.get('target_segment') in self.segments:
            segment_modifier = 1.2
        
        # Итоговая вероятность
        probability = base_prob * barrier_modifier * engagement_modifier * segment_modifier
        
        return min(max(probability, 0), 1)  # Ограничение [0, 1]
    
    def simulate_response_distribution(self, intervention: dict, n: int = 1000) -> list:
        """
        Симуляция распределения откликов для статистического анализа
        """
        responses = []
        for _ in range(n):
            base_prob = self.predict_response_probability(intervention)
            # Добавляем случайный шум (модель ошибки)
            noise = np.random.normal(0, 0.1)
            response = 1 if base_prob + noise > 0.5 else 0
            responses.append(response)
        return responses
```

## 5.1.4. Алгоритм симуляции

```python
def run_intervention_simulation(twins: list, interventions: list, n_runs: int = 1000):
    """
    Запуск симуляции интервенций на популяции Twin
    
    Args:
        twins: список Digital Twin
        interventions: список интервенций для тестирования
        n_runs: количество запусков симуляции
        
    Returns:
        results: словарь с результатами по каждой интервенции
    """
    results = {}
    
    for intervention in interventions:
        all_responses = []
        
        for twin in twins:
            model = TwinResponseModel(twin.attributes)
            
            # Симуляция множественных запусков
            run_responses = model.simulate_response_distribution(
                intervention, n_runs
            )
            all_responses.extend(run_responses)
        
        # Статистика по интервенции
        results[intervention['id']] = {
            'mean_response_rate': np.mean(all_responses),
            'std': np.std(all_responses),
            'ci_95': stats.t.interval(0.95, len(all_responses)-1,
                                      loc=np.mean(all_responses),
                                      scale=stats.sem(all_responses)),
            'distribution': np.histogram(all_responses, bins=10)
        }
    
    return results
```
