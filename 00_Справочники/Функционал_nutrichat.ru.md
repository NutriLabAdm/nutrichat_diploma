# Функционал Нутричат (из кода)

Версия: 1.0 | Дата: 18.04.2026

---

## Обзор платформы

Нутричат — платформа персонализированного нутрициологического сопровождения с AI-анализом питания по фотографиям.

---

## Модули анализа

### 1. Нутрициологический анализ (Food Recognition)

**Уровни анализа:**

| Уровень | Название | Описание |
|---------|----------|----------|
| 1 | Overview (Обзор) | Базовый анализ блюда, калории, БЖУ |
| 2 | Professional (Профессиональный) | Детальный анализ с оценкой по Гарвардской тарелке |
| 3 | Bio (Биохакер) | Глубокий анализ биодоступности, синергий, антинутриентов |

**Ключевые метрики:**

- **NDI (Nutrient Density Index)** — индекс нутриционной плотности
- **Harvard Plate Assessment** — оценка по Гарвардской тарелке
- **Health Score** — общая оценка здоровья блюда
- **Glycemic Load** — гликемическая нагрузка
- **Satiety Index** — индекс насыщения

### 2. Красные флаги (Red Flags)

Автоматическое определение негативных факторов:
- Переработанное мясо
- Избыток жиров
- Майонез и соусы
- Высокая калорийность
- Избыток натрия

---

## Функционал дневника питания

### Добавление записей

- Ручной ввод продуктов
- Сканирование штрих-кодов
- Распознавание по фотографии (CV-модуль)
- Уточняющие вопросы AI

### Пересчёт блюд (Recalc)

Пользователь может уточнить:
- Замену ингредиентов (напр: "без майонеза")
- Изменение веса (напр: "порция 200г")
- Способ приготовления (напр: "отварное вместо жареного")
- Цель (похудеть, набрать массу, ЗОЖ)

## AI-рекомендации

### Типы рекомендаций

| Категория | Пример |
|-----------|--------|
| **Добавить** | Добавьте больше овощей |
| **Уменьшить** | Уменьшите жиры |
| **Заменить** | Отварное вместо жареного |
| **Улучшить сочетание** | Для усвоения железа добавьте витамин C |

### Персонализация под цели

- Похудение (weight_loss)
- Набор массы (muscle_gain)
- Здоровое питание (health)
- Метаболическое здоровье (metabolic_health)
- Долголетие (longevity)

## Визуализации

### Harvard Radar (Радар Гарварда)

Полярная диаграмма оценки:
- Белки
- Овощи
- Углеводы
- Жиры
- Клетчатка
- Нутри-индекс
- Приготовление

### Таблица нутриентов

Детальная таблица с:
- Нутриент (белки, жиры, углеводы, витамины, минералы)
- Источники
- Количество
- % от дневной нормы
- Рекомендация

## Техническая реализация

### API endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /api/food/analyze | Анализ блюда по фото |
| POST | /api/food/recalc | Пересчёт с уточнениями |
| POST | /api/logs | Логирование приёмов пищи |

- `FOOD_OVERVIEW_PROMPT` — базовый анализ
- `NUTRITION_ANALYSIS_PREMIUM_PROMPT` — профессиональный анализ
- `NUTRITION_BIOHACKER_PROMPT` — биохакерский анализ

## Пользовательский путь (User Journey)

### Основной сценарий: Анализ блюда

```
1. Пользователь фотографирует еду
       ↓
2. Загружает фото в приложение
       ↓
3. AI анализирует изображение
       ↓
4. Отображает результат:
   - Название блюда
   - Ингредиенты
   - Калории и БЖУ
   - Оценка по Гарварду
   - Красные флаги
       ↓
5. Пользователь может:
   - Добавить в дневник
   - Уточнить состав (Recalc)
   - Получить рекомендации
```

### Сценарий: Пересчёт блюда

```
1. Пользователь открывает карточку блюда
       ↓
2. Нажимает "Уточнить" или меняет параметры
       ↓
3. Вводит изменения:
   - Замена ингредиентов
   - Изменение веса
   - Новый способ приготовления
   - Другая цель
       ↓
4. AI пересчитывает:
   - Новые нутриенты
   - NDI оценку
   - Рекомендации
       ↓
5. Пользователь принимает решение
```

## Экраны приложения

### Главный экран анализа блюда

1. **Изображение блюда** — фото еды (центральный элемент)
2. **Теги приёма пищи** — Завтрак/Обед/Ужин/Перекус
3. **Название блюда** — "Сосиски с оливье"
4. **Ингредиенты** — теги отдельных продуктов
5. **Табы переключения** — Обзор | Анализ | Bio
6. **Краткая статистика** — NDI, Ккал, Белки, Жиры, Углеводы
7. **Harvard Radar** — радарная диаграмма
8. **Оценка Гарварда** — текстовая рекомендация
9. **Секция пересчёта** — форма уточнений + кнопки-подсказки
10. **Кнопка добавления** — "+ В дневник"

### Панель анализа (правая колонка)

- Название блюда
- **Красные флаги** (секция предупреждений)
- Таблица нутриентов:
  - Макронутриенты (белки, жиры, углеводы, калории)
  - Витамины (A, C, D, B-комплекс)
  - Минералы (железо, кальций, магний, калий)
- Сочетание продуктов
- Приготовление
- Плюсы блюда
