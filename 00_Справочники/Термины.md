# Справочник терминов

Версия: 1.0 | Дата: 18.04.2026

---

## Основные термины платформы

| Термин | Определение | Примечание |
|--------|-------------|------------|
| Нутричат (NutriChat) | Платформа персонализированного нутрициологического сопровождения | Основное название продукта |
| Профиль (Profile) | Статические данные пользователя: рост, вес, цели, ограничения | Ядро системы |
| Цифровой двойник (Digital Twin) | Динамическая модель поведения пользователя на основе ML | Расширение профиля |
| Запись питания (FoodEntry) | Запись о потреблённом продукте с нутриентами | Основная сущность трекинга |
| Цель (Goal) | Целевой параметр пользователя (вес, калории, нутриенты) | |
| Энциклопедия здоровья (Health Encyclopedia) | База знаний о продуктах, нутриентах, витаминах, рецептах | Ранее "Нутриэнциклопедия" |
| Медиа-библиотека (Media Library) | Хранение фото, видео, документов пользователя | |
| Биллинг (Billing) | Управление подписками и платежами | |
| Трекинг (Tracking) | Отслеживание метрик: вес, активность, настроение | |

## Архитектурные термины

| Термин | Определение |
|--------|-------------|
| Микросервис | Независимый сервис, решающий одну задачу |
| API (Application Programming Interface) | Программный интерфейс взаимодействия |
| Container Registry | Хранилище контейнеров (Docker) |
| CI/CD | Непрерывная интеграция и доставка |
| Kubernetes (K8s) | Платформа оркестрации контейнеров |
| S3 | Объектное хранилище (AWS) |
| PostgreSQL | Реляционная база данных |
| Redis | In-memory кэш |
| RabbitMQ | Очередь сообщений |
| GraphQL | Язык запросов к API |
| gRPC | Высокопроизводительный RPC |
| WebSocket | Протокол двусторонней связи |
