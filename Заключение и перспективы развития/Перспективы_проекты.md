# Перспективные проекты (Дополнительные модули)

В данном разделе представлены перспективные направления развития экосистемы Нутричат, находящиеся на стадии проектирования.

## Перспективные модули

### Коммуникации
- **coach.nutrichat.ru** — Связь с нутрициологами
  - Видеоконсультации с диетологами
  - Персональные планы питания
  
- **community.nutrichat.ru** — Сообщество пользователей
  - Групповые челленджи
  - Поддержка единомышленников

### Персонализация
- **genes.nutrichat.ru** — Генетический тест
  - Анализ генетических маркеров метаболизма
  - Персонализированные рекомендации на основе ДНК
  
- **gut.nutrichat.ru** — Анализ микробиома
  - Тестирование кишечной микрофлоры
  - Рекомендации по питанию для здоровья кишечника

### Интеграции с устройствами
- Умные весы (Xiaomi, Withings)
- Фитнес-трекеры (Fitbit, Xiaomi Mi Band)
- Глюкометры (для пользователей с диабетом)
- Смарт-кольца (Oura, RingConn)
